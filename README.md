# react-router-redux-ie8
react-router-redux for ie8

## Installation
`$ npm install react-router-redux-ie8 -S`

## Usage
```
// webpack.config.js
{
  resolve: {
    alias: {
      'react-router-redux': 'react-router-redux-ie8'
    }
  }
}
```
Or Use `import react-router-redux-ie8` instead `import react-router-redux`
